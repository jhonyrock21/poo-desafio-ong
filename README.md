# Sistema para gestão de ONG

Atividade prática de Programação Orientada a Objetos em Java.

O projeto simula uma parte simples de um sistema de ONG. Foram usadas classes para representar membros da ONG, voluntários, doadores e projetos sociais.

## Conceitos usados:
- Encapsulamento
- Herança
- Sobrescrita de métodos
- Composição

## Classes:
- MembroONG
- Voluntario
- Doador
- ProjetoSocial
- SistemaMain
